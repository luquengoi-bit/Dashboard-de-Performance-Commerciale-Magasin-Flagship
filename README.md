# Dashboard-de-Performance-Commerciale-Magasin-Flagship

## Objectif du projet

Analyse des performances commerciales d’un magasin flagship à partir d’indicateurs clés comparés entre l’année N et N-1.

## Outils utilisés

- Microsoft Excel
- Tableaux croisés dynamiques
- KPI
- Analyse comparative N / N-1
- Visualisation de données

## Indicateurs analysés

- Chiffre d’affaires
- Coût d’achat
- Marge
- Nombre de ventes
- Panier moyen
- Contribution du secteur vêtements au CA global

## Principaux constats

- Le chiffre d’affaires global progresse de +2,70%
- Le secteur vêtements représente près de 20% du CA total
- La marge du secteur vêtements est en baisse malgré la hausse du volume
- Le panier moyen diminue, suggérant une stratégie promotionnelle

## Recommandations business

- Réévaluer la politique promotionnelle
- Optimiser les produits à forte marge
- Suivre les ventes à faible rentabilité

## Compétences démontrées

- Analyse de performance commerciale
- Lecture de KPI
- Interprétation business des données
- Création de tableaux de bord Excel
